Vibe Coding 이란?

Vibe Coding은 복잡한 문법이나 로직에 얽매이지 않고, 개발자가 구현하고자 하는 의도(Intent)와 분위기(Vibe)를 AI에게 전달하여 소프트웨어를 완성하는 차세대 개발 방식입니다.

Vibe Coding의 장점

🗣️

자연어 중심의 의도 전달

🤖

AI의 자율적 코드 생성

⚡

개발 진입 장벽의 붕괴

🏗️

초고속 프로토타이핑

Vibe Coding의 원칙

1

3

테스트 코드 작성을 통해 잘 구현되었는지 확인합니다.

Gemini CLI

터미널 기반의 강력한 AI 에이전트

Gemini CLI 란?

터미널 환경에서 Google의 강력한 AI 모델과 직접 대화하고 텍스트 생성, 코드 분석, 시스템 자동화를 수행할 수 있는 명령어 라인 인터페이스 도구입니다.

Gemini CLI의 특별한 기능

프롬프트 기반 Spec 설계

사용자의 막연한 아이디어를 체계적인 개발 문서(요구사항, 설계안)로 자동 변환합니다.

멀티파일 코드 생성

하나의 프롬프트로 여러 파일을 포함한 전체 프로젝트 구조를 한 번에 생성합니다.

시스템 인스트럭션 제어

.txt 또는 .md 파일을 통해 AI의 작업 방식과 코딩 표준을 강제로 스티어링합니다.

터미널 통합 자동화

환경 구축(npm install 등)부터 실시간 디버깅까지 터미널과 직접 상호작용합니다.

지능형 컨텍스트 참조

@file, #codebase 지시어로 파일 시스템 전체 맥락을 이해한 답변을 제공합니다.

배포 자동화 워크플로우

로컬에서 완성된 코드를 Firebase나 Cloud Run 등으로 즉시 배포하는 스크립트를 실행합니다.

직접 수행하는 AI

AI가 코드 수정, 테스트 실행, 프로젝트 분석을 직접 수행하는 '에이전트' 역할을 합니다.

Google Code Assist

IDE 기반의 지능형 코딩 파트너

Google Code Assist 란?

VS Code, JetBrains 등 즐겨 쓰는 IDE에서 직접 동작하며, 대규모 코드베이스와 클라우드 환경에 최적화된 개발 지원을 제공하는 AI 보조 도구입니다.

Code Assist의 주요 기능

실시간 AI 코드 완성

코드 작성 중 AI가 다음 코드를 예측하여 제안하며, 자연어 주석만으로 기능을 구현합니다.

코드 이해 및 리팩터링

복잡한 로직을 설명하고 성능 개선, 가독성 향상을 위한 수정안을 즉시 제시합니다.

클라우드 최적화 지원

Google Cloud 서비스 연동 코드 및 인프라(IaC) 구성을 모범 사례 기반으로 생성합니다.

엔터프라이즈 보안 및 관리

조직 내 코드 스타일 준수, API 키 보호 등 기업 환경에 필수적인 거버넌스를 제공합니다.

보조하는 AI

개발자가 중심이 되고, AI가 실시간으로 제안 및 리뷰를 수행하는 '파트너' 역할을 합니다.

Gemini CLI 설치 가이드

1 Google Cloud CLI 설치

Windows

설치 프로그램 다운로드

macOS

Homebrew 사용 시:

brew install --cask google-cloud-sdk
직접 다운로드 설치 가이드

2 Node.js 설치 (LTS 권장)

Windows

Installer 다운로드 (.msi)

macOS

brew install node
pkg 다운로드 설치

3 Gemini CLI 설치

터미널(또는 CMD/PowerShell)에서 아래 명령어를 실행하세요.

npm install -g @google/gemini-cli

※ Mac/Linux에서 권한 오류 발생 시 앞에 sudo 를 붙여주세요.

4 환경 설정 및 프로젝트 선택

1) Google Cloud 프로젝트 ID 확인

Google Cloud Console의 대시보드에서 '프로젝트 ID'를 복사하세요.

GCP 콘솔 대시보드 바로가기

2) .env 파일 생성 및 ID 입력

실습 프로젝트의 루트(최상위) 디렉토리에 .env 파일을 만들고 복사한 ID를 넣으세요.

GOOGLE_CLOUD_PROJECT="여기에-복사한-ID-입력"
GOOGLE_CLOUD_LOCATION="global"

3) Google Cloud 인증

gcloud auth login

5 VS Code 설치

Windows/Mac 공용

VS Code 다운로드

macOS (Homebrew)

brew install --cask visual-studio-code

6 Gemini CLI Companion 설치

VS Code에서 에이전트를 더 효율적으로 사용하기 위한 확장 프로그램을 설치합니다.

설치 방법:

  1. VS Code 왼쪽 메뉴의 Extensions(확장) 아이콘 클릭
  2. 검색창에 'Gemini CLI Companion' 입력
  3. 파란색 Install 버튼 클릭

7 Gemini 실행 및 인증

이제 모든 준비가 끝났습니다! 터미널에서 Gemini를 실행하고 인증을 완료하세요.

실행 및 인증 단계:

  1. VS Code 상단 메뉴에서 Terminal > New Terminal 선택
  2. 터미널에 gemini 입력 후 엔터
  3. Gemini 프롬프트가 뜨면 /auth를 입력하고 엔터
  4. 화살표 키를 사용하여 Vertex AI를 선택하고 엔터
  5. 자동으로 열리는 웹 브라우저에서 Google 계정으로 로그인
# 1. Gemini 실행
gemini
? How can I help you today?

# 2. 인증 명령어 입력
/auth
? Select an authentication method:
❯ Vertex AI
Google AI (Gemini API)

※ 로그인이 완료되면 이제 대화와 코딩 실습을 시작할 수 있습니다!

  1. VS Code 하단 상태 표시줄의 Gemini Code Assist (또는 Cloud Code) 클릭

  3. 목록에서 실습에 사용할 프로젝트 ID를 선택하세요.

용어 설명

자연어 코딩 (Vibe Coding)

“AI에게 자연어로 지시하면, AI가 코드를 대신 작성하고 실행해주는 방식"입니다. 마치 내가 원하는 기능을 말하면 직원이 대신 만들어주는 것과 같습니다.

코드 편집 화면 (IDE)

Integrated Development Environment의 줄임말로, 코딩, 디버깅, 컴파일, 배포 등 소프트웨어 개발에 필요한 모든 도구를 하나로 통합한 개발 환경입니다.

API (Application Programming Interface)

API는 프로그램끼리 도와주고, 말 잘 통하게 해주는 약속된 언어입니다. 복잡한 걸 몰라도, 메뉴판처럼 버튼 하나로 필요한 걸 부탁할 수 있게 해줍니다.

보일러플레이트

반복적으로 작성해야 하는 기초적인 코드 조각들을 의미합니다.

레거시 코드

과거에 작성되어 현재 유지보수 중인 오래된 코드를 의미합니다.

기술 부채

빠른 개발을 위해 선택한 임시방편적인 코드가 나중에 수정 비용으로 돌아오는 것을 의미합니다.

CI/CD

개발한 코드를 자동으로 검사하고 배포하는 자동화 시스템입니다.

TTM (Time-to-Market)

아이디어가 실제 제품으로 출시되기까지 걸리는 시간을 의미합니다.

목업 (Mockup)

실제 제품이 만들어지기 전에 미리 시각적으로 확인해 볼 수 있는 모형입니다.

개인화 추천 마케팅의 정석

스타벅스는 AI 추천 알고리즘을 활용, 고객의 구매 이력과 날씨 등 외부 데이터를 결합해 개인 맞춤형 음료를 제안했다. 그 결과 이메일 오픈율은 3배 상승했으며, 앱 이용률 또한 크게 증가하는 성과를 거뒀다.

BMW – 생성형 AI, 광고계에 새 바람

타겟 맞춤형 비주얼 광고 자동 생성

BMW는 생성형 AI 기술을 광고 제작에 도입했다. 고객 타겟에 맞춰 자동차 이미지와 광고 카피를 자동으로 생성하는 캠페인을 통해 소셜미디어에서의 전환율을 2배 이상 끌어올리는 데 성공했다.

Nutella – 세상에 단 하나뿐인 700만 개의 디자인

알고리즘이 만든 패키징, 완판 신화 쓰다

L'Oréal – AI가 글로벌 마케팅 속도 높여

자연어처리(NLP)로 콘텐츠 제작 시간 80% 단축
